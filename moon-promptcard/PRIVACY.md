# 隐私政策 · Moon PromptCard

最后更新：2026-06

Moon PromptCard（以下简称「本插件」）重视你的隐私。本政策说明本插件如何处理数据。

## 1. 我们读取什么
本插件**仅在你主动触发分析时**读取以下内容之一：
- 你在网页中**选中的文本**；
- 或你当前**聚焦输入框**（`textarea` / `input` / `contenteditable`）中的文本。

本插件**不会**自动收集整页内容、不会记录键盘输入、不会读取浏览历史或其它页面数据。

## 2. 数据如何被处理
取决于你选择的服务模式：

- **自定义 API 模式**：被分析的提示词文本会发送到**你自己配置的接口地址**（Base URL），使用你自己的 API Key。本插件不经手、不存储你的请求内容，也不会把这些内容发送到我们的服务器。
- **内置服务模式**：被分析的提示词文本会发送到本插件**官方后端**，用于完成认证、计次与模型调用并返回分析结果。我们仅为提供该功能而处理这些文本。

## 3. 本地存储
以下信息保存在你浏览器的 `chrome.storage.local`，仅存于本地设备：
- 界面 / 提示词语言、主题、悬浮球开关；
- 服务模式；
- 自定义 API 的 Base URL、API Key、模型名称；
- 最近一次及历史分析结果。

**你的自定义 API Key 只存储在本地，永远不会上传到我们的服务器。** 内置服务的登录凭证与自定义 API Key 分开存储。

## 4. 你的控制权
- 在设置页可随时切换语言、主题、退出登录。
- 你可以在 `chrome://extensions` 移除本插件，或清除浏览器存储，以删除全部本地配置与历史记录。

## 5. 第三方
本插件不向任何第三方出售或转移你的数据。自定义 API 模式下，数据流向完全由你配置的接口决定。

## 6. 远程代码
本插件不下载或执行任何远程代码，不使用 `eval`。

## 7. 联系方式
如对本政策有疑问，请通过 Chrome Web Store 商品页提供的开发者邮箱联系我们。
