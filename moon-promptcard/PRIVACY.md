# 隐私政策 · Moon PromptCard

最后更新：2026-06-29

Moon PromptCard（以下简称「本插件」）重视你的隐私。本政策说明本插件如何处理数据。

## 1. 我们读取什么
本插件**仅在你主动触发操作时**读取以下内容之一：
- 你在网页中**选中的文本**；
- 你当前**聚焦输入框**（`textarea` / `input` / `contenteditable`）中的文本；
- 你**主动右键选择**的网页图片，或你在面板中**主动上传**的图片（用于"图片转提示词"或图生图）。

本插件**不会**自动收集整页内容、不会记录键盘输入、不会读取浏览历史或其它页面数据。

## 2. 数据如何被处理
取决于你选择的服务模式：

- **自定义 API 模式**：被分析的提示词文本、用于"图片转提示词/图生图"的图片，会发送到**你自己配置的接口地址**（Base URL），使用你自己的 API Key。本插件不经手、不存储这些内容，也不会发送到我们的服务器。
- **内置服务模式**：你需用邮箱验证码登录。被分析的提示词文本（以及"图片转提示词"时的图片）会发送到本插件**官方后端**，由后端调用模型完成分析并返回结果，同时按账号计次。我们仅为提供该功能而处理这些数据，不用于其它用途、不出售。

文生图 / 图生图功能**始终使用你在设置中心自配的接口与 Key**，直接由你的浏览器请求你配置的服务，不经过我们的服务器。

## 3. 本地存储
以下信息保存在你浏览器的 `chrome.storage.local`，仅存于本地设备：
- 界面 / 提示词语言、主题、悬浮球开关、服务模式；
- 自定义网关的 Base URL、API Key、Chat / 文生图 / 图生图模型、默认尺寸与数量；
- 内置服务的登录令牌与账号；
- 最近一次及历史分析结果。

**你的自定义 API Key 只存储在本地，永远不会上传到我们的服务器。** 内置服务的登录令牌与自定义 API Key 分开存储。

## 4. 内置服务后端会收到什么
当且仅当你使用内置服务模式：
- 登录时：你的邮箱地址（用于发送验证码、建立账号与计次）；
- 分析 / 读图时：你提交的提示词文本或图片（用于生成结果），以及你的登录令牌。
后端不存储你的浏览历史，不读取你未主动提交的内容。

## 5. 你的控制权
- 在设置页可随时切换语言、主题、服务模式、退出登录、关闭悬浮球。
- 你可以清空设置中心的网关配置（包含 API Key）。
- 你可以在 `chrome://extensions` 移除本插件，或清除浏览器存储，以删除全部本地配置与历史记录。

## 6. 权限用途
- `storage`：保存上述本地配置与分析结果。
- `activeTab` + `scripting`：在你操作时读取选中文本 / 聚焦输入框内容，并把优化结果写回。
- `contextMenus`：提供"分析选中文本""把图片转成提示词"右键菜单。
- `host_permissions`（`http://*/*`、`https://*/*`）：用于把你配置的网关请求、或内置后端请求发送出去；并允许悬浮分析条在任意网站可用。**插件默认不读取任何页面内容，仅在你主动点击时才读取。**

## 7. 远程代码
本插件不下载或执行任何远程代码，不使用 `eval`，不混淆代码。

## 8. 第三方
本插件不向任何第三方出售或转移你的数据。自定义 API / 图片功能下，数据流向完全由你配置的接口决定。

## 9. 联系方式
如对本政策有疑问，请通过 Chrome Web Store 商品页提供的开发者邮箱联系我们。
