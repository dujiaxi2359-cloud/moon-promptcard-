# Chrome Web Store 上架资料 · Moon PromptCard

> 本文件是上架时要填的所有内容，照着粘贴即可。Chrome 后台禁止自动化，需你手动操作。

## 插件名称
Moon PromptCard

## 简短描述（≤132 字符）
悬浮式 AI 提示词助手：对输入框/选中文本评分、诊断、一键优化，支持图片转提示词、文生图/图生图，自带 OpenAI 兼容网关配置。

## 详细描述
Moon PromptCard 是一个浏览器悬浮式提示词分析与优化工具，面向使用图像 / 视频生成模型（Veo、Sora、Kling、Seedance、Omni、GPT Image、Gemini Image、Midjourney、Stable Diffusion 等）的 AIGC 创作者。

核心功能：
- 提示词分析：在任意网页输入框写提示词，或选中一段文本，点悬浮按钮即可获得评分、问题诊断、优化建议、优化后的完整提示词与负面限制；可一键复制或替换原文。
- 图片转提示词：右键任意网页图片（或在面板上传图片），用视觉模型反推出可直接使用的中 / 英文提示词。
- 文生图 / 图生图：在设置中心填入你自己的图像网关，即可从提示词生成图片，或基于参考图生成（含「混合生成 4 张」预设）。
- 两种分析服务模式：
  - 内置服务：邮箱登录后由官方后端完成分析（按账号计次）。
  - 自定义 API：填写你自己的 Base URL / Key / 模型，直接调用任意 OpenAI 兼容网关，配置只存本地。

分析维度涵盖：主体 / 产品 / 人物是否固定、场景与动作是否清楚、镜头语言、风格统一性、画幅 / 时长 / 比例、是否适配目标模型、冲突词、负面限制缺失、变脸 / 变形 / 文字乱码风险、是否可直接生成等。

界面：黑橙「Eclipse」主题，玻璃拟态结果卡，深 / 浅色网页都清晰。

## 单一用途（Single purpose）
帮助用户分析与优化 AI 生成类提示词，并基于其配置的接口进行图片相关的提示词生成。

## 权限说明（逐条对应后台「权限理由」）
- `storage`：在本地保存语言 / 主题 / 服务模式、网关配置、登录令牌与最近分析结果。
- `activeTab` + `scripting`：仅在用户主动点击时，读取当前标签页的选中文本或聚焦输入框内容，并把优化结果写回输入框。
- `contextMenus`：提供「分析选中文本」「把图片转成提示词」右键菜单。
- `host_permissions`（`http://*/*`、`https://*/*`）：① 让悬浮分析条可在任意网站调出；② 将请求发送到用户在设置中配置的网关、或内置服务后端。**默认不读取任何页面内容，仅在用户主动操作时读取。**

### 为什么需要广泛 host 权限
提示词输入框可能出现在任意网站；且用户可自配任意 OpenAI 兼容网关地址，扩展需能向用户指定的任意 https 主机发起请求。插件不自动采集整页 DOM、不记录键盘、不读浏览历史。

## 是否有内购
有（内置服务的分析次数 / 套餐），通过外部支付页完成，不在插件内处理支付。

## 是否使用远程代码
否。所有 JavaScript 随扩展打包，运行时不下载 / 不执行远程脚本，不使用 `eval`，不混淆。

## 数据使用声明（Data usage）
- 会处理：用户**主动提交**的提示词文本、用于"图片转提示词/图生图"的图片；内置服务模式下还包括登录邮箱。
- 用途：仅用于生成分析 / 图片结果与账号计次。
- 不收集：浏览历史、整页内容、键盘记录。
- 自定义 API / 图片功能：数据只发往用户自己配置的接口，不经官方服务器。
- 不向第三方出售或转移用户数据。
- 勾选项参考：This item does NOT sell/transfer user data to third parties；用途选「App functionality」「Account management」。

## 上架操作步骤（你手动）
1. 打开 chrome.google.com/webstore/devconsole → 你的开发者账号。
2. 「上传新内容」→ 上传 `moon-promptcard-upload.zip`（manifest 在 zip 根目录）。
3. 「商品详情」：粘贴上面的名称、简短描述、详细描述、单一用途；语言选中文（简体）+可加英文。
4. 上传图标、≥1 张截图、（可选）宣传图。
5. 「隐私权」标签：填隐私政策 URL；逐条填写权限理由（用上面的"权限说明"）；远程代码选「否」；按"数据使用声明"勾选。
6. 「分发」：地区、可见性（先 Unlisted 测试，再转 Public）。
7. 保存草稿 → 「提交审核」。审核通常数小时到数天。
